作业需求：
# 三级菜单：
# 1. 运行程序输出第一级菜单
# 2. 选择一级菜单某项，输出二级菜单，同理输出三级菜单
# 3. 菜单数据保存在文件中


介绍：
三级菜单，想尽量把功能抽象出来，最终结果不是很理想，还是叠加了大量代码。通过嵌套循环来实现不同级别切换
main.py:
    manage_list方法：将无序的字典key添加序号后存入列表，并用两层循环进行格式化输出
    option方法，根据接收值取到相应的key，以取出对应的value

menu.txt：存储数据，利用嵌套字典的方式完成，最内层使用列表存储
    数据格式：

    {'华北':
        {'北京':
            {'东城':
                ['东直门', '雍和宫'],'
            }
        }
    '华南':.......
    }
