# 购物车：
# 1. 商品信息- 数量、单价、名称
# 2. 用户信息- 帐号、密码、余额
# 3. 用户可充值
# 4. 购物历史信息
# 5. 允许用户多次购买，每次可购买多件
# 6. 余额不足时进行提醒
# 7. 用户退出时 ，输出档次购物信息
# 8. 用户下次登陆时可查看购物历史
# 9. 商品列表分级显示

-------------------------------------------------------------
user.txt

    用户信息表，字典类型：
        {'用户名': ['密码', 锁定状态（1锁定，0解锁）, 密码错误次数（大于2不能登陆）]}

commodity.txt

    商品详情表：格式如下
        {'汽车':
                {'法拉利': [单价, 库存],
                 '宝马': [200000, 48],
                 '奔驰': [300000, 50]
                 },
         '家具':.....
         }

user_money.txt

    {'用户名': 余额}

shopping_cart.txt
    购物车列表

    {'用户名':
        {'商品名': [单价, 数量]},
    '用户名':
        {'商品名': [单价, 数量]},
    }

shopping_histry.txt
    购物历史记录表

    {'用户名':[
        [time,{'商品名': [单价, 数量]}],[time,{'商品名': [单价, 数量]}],
    '用户名':[
        [time,{'商品名': [单价, 数量]}],[time,{'商品名': [单价, 数量]}],
    ]}

file_opt.py

    读写文件的类，可以操作user.txt，commodity.txt，user_money.txt，shopping_cart.txt四张表

main.py

    主函数，最外侧一个死循环，负责打印菜单接收用户输入，控制流程
